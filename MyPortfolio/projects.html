Tic Tac Toe Game

A simple and fun game that can be played by two players on the same device.The Tic Tac Toe Clone project is a simple yet engaging game that mimics the classic Tic Tac Toe game. It involves two players taking turns to place their markers (traditionally X and O) on a 3x3 grid, with the objective of getting three of their markers in a row, either horizontally, vertically, or diagonally. This project is an excellent opportunity to practice programming concepts such as loops, conditionals, and user input handling, making it ideal for beginners looking to enhance their skills in a fun and interactive way.

Skills used: HTML, CSS
